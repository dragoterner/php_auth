# php_auth
Понятный PHP-скрипт, который проверяет ID пользователя через GET, сессию или куки и возвращает контент или 403 Forbidden. В примере тоже добавлена базовая защита от перебора и комментарии. По желанию могу адаптировать под конкретный стек или усилить защиту

Реализован простой PHP-скрипт, который:
получает ID пользователя из трёх источников: GET-параметра (uid), сессии или cookie;
сравнивает полученный ID с заранее заданным разрешённым ID;
возвращает контент только при совпадении, иначе возвращает 403 Forbidden.

Добавлена базовая защита от перебора попыток доступа:
счётчик попыток хранится в сессии;
после превышения лимита (5 попыток) возвращается 403 Forbidden.
Скрипт снабжён комментариями для понятности.
Пример легко адаптировать под продакшн: HTTPS, расширенная защита, полноценная авторизация.

Как использовать
Добавить файл с PHP-скриптом в проект.
Указать разрешённый ID в константе/переменной внутри скрипта.
При необходимости изменить источник идентификатора или лимит попыток.
